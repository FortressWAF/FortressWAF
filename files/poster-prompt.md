# Brutalism Poster Prompt for ChatGPT (DALL-E)

Salin teks di bawah ini ke ChatGPT:

---

Design a brutalism-style tech poster for an open-source project called "FortressWAF" (Enterprise WAF & API Security Gateway).

**Style:** Brutalism / Neo-Brutalism
- Bold, heavy sans-serif typography (think Helvetica Bold, Impact)
- High contrast black and white with one accent color (bright red #FF0000 or neon orange)
- Raw, industrial aesthetic
- Asymmetric grid layout
- Harsh shadows, thick borders, overlapping elements
- Distorted or glitch effects on text
- Monochromatic base with aggressive red accent

**Elements to include:**
- Title: "FORTRESSWAF" in massive bold letters, possibly cut off at edges
- Subtitle: "Enterprise WAF & API Security Gateway"
- Badge: "OPEN SOURCE — v1.4.0"
- Tagline: "JA3 · BEHAVIORAL · WASM · DESYNC · ADAPTIVE · eBPF"
- Tagline 2: "PARSER HARDENING · FP REDUCTION · PERFORMANCE ISOLATION"
- CTA: "WE NEED CONTRIBUTORS" in red with a glitch effect
- Stars graphic: star icons with "★ CONTRIBUTE ON GITHUB ★"
- URL: "github.com/FortressWAF/FortressWAF" in monospace at bottom

**Layout:**
- A4 / 1080x1920 vertical format (for Instagram/LinkedIn)
- Heavy black borders, 20px minimum
- Text overlapping other text
- Jagged or cut edges
- Gravel/grain texture overlay
- QR code placeholder (a black square with white pixels in bottom-right)

**Mood:**
- Aggressive, raw, hacker aesthetic
- Feels like a punk rock flyer from the 90s but for cybersecurity
- Industrial, underground, anti-corporate vibe

**Color palette:**
- Background: #1a1a1a (dark charcoal) or #ffffff
- Primary text: #ffffff on dark, #000000 on light
- Accent: #ff0000 (pure red) or #ff4500 (orange-red)
- Secondary: #cccccc or #333333

Generate the image in 1080x1920 ratio.
