### LINKEDIN — First Post

```
Just shipped something I've been building for the past few months — FortressWAF.

It's a production-grade Web Application Firewall built to actually compete with Cloudflare WAF, Imperva, and Fortiweb. But here's the thing: most enterprise WAFs cost a fortune and take days to configure. FortressWAF deploys in under 30 minutes and the core is open-source.

What it does:
→ Blocks OWASP Top 10 attacks in real-time (SQLi, XSS, RCE, you name it)
→ ML-powered anomaly detection catches zero-days that signature-based rules miss
→ Built-in DDoS protection, bot management, API security
→ Virtual patching — deploy CVE patches without touching your app code
→ Real-time dashboard with live attack visualization

The stack: Go for the proxy core, Python/FastAPI for the ML sidecar, Next.js dashboard.

Why I built it: watched one too many companies get hit by Log4Shell because their WAF setup was too complex to tune properly. Simplicity was the goal without sacrificing security.

It's not a POC. It's deploy-in-production ready.

GitHub: https://github.com/zulfff/FortressWAF

Would love feedback from the security community.
```
